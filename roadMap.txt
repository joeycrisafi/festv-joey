## Deployment gotchas (hard-won)
	- CORS must be scoped to /api only — global CORS returns 500 for JS/CSS asset requests → blank page
	- Render build command must NOT end with npm run build — triggers second frontend build with emptyOutDir:true that wipes react-dist
	- framer-motion must stay at v10 — v12 WAAPI bug skips initial state on mount
	- AnimatePresence initial={false} kills ALL descendant animations — never use it
	- Render Build Filters must include frontend/** or frontend-only commits don't trigger a deploy
	- react-dist is gitignored — never commit build artifacts; Render generates on every deploy
