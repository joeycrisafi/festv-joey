## Decisions log
	- Model: using claude-haiku-4-5 (new Anthropic accounts only have Claude 4 access in 2026). Upgrade to sonnet when tier allows.
	- Favorites: saves to both localStorage (event planning flow) and API (cross-device). Both in sync.
	- Browse vendors: API-only, no static fallback. If DB is empty, shows empty state.


## Known risks / open questions
	- Vendor search currently returns ALL providers (not VERIFIED only) — flip before launch
	- No Cloudinary yet — vendor profile images use avatarUrl from DB or Unsplash fallbacks
	- Stripe not connected — deposit flow UI is built but payment not wired yet
